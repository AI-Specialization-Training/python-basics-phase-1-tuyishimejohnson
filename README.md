# Python Basics Exercises

**Welcome!** 🎉  

This set of exercises is **NOT a test or graded assignment**.  
It is simply an **opportunity to solidify** the Python basics you’ve already covered. 
Think of it as practice reps in the gym — the more you do it yourself, the stronger your understanding becomes.

---

### 🚨 IMPORTANT RULES (Please read before starting)

1. **Disable AI autocomplete in your IDE**  
   - **VS Code**: Go to Settings → Search “inline suggest” → Turn OFF “Inline Suggest: Enable”  
   - **PyCharm**: Settings → Editor → General → Code Completion → Uncheck “Show suggestions as you type”  
   - **Why?** We want your brain to do the thinking, not the AI.

2. **No AI tools allowed at all**  
   - Do **not** use ChatGPT, Grok, Claude, GitHub Copilot, Cursor, etc.  
   - This includes asking AI to “explain” or “write” any part of the solution.

3. **Google methods, NOT solutions**  
   - ✅ Allowed: “python how to check if string contains substring” or “python list append vs extend”  
   - ❌ Not allowed: “solution to exercise 3 python basics” or pasting your code into Google  
   - The goal is to learn **how to find information** like a real programmer.

---

### How to Complete the Exercises

1. Open the project in your IDE (VS Code or PyCharm).
2. Work through the files **in order**:
   - `exercise_01_...`
   - `exercise_02_...`
   - etc.
3. Each file has:
   - Instructions at the top (in comments)
   - `# TODO` comments where you need to write code
4. When you finish an exercise, move to the next one.  
   **Do not skip ahead** 
5. When you are done with one exercise commit your changes and at the end of all exercises you can push all you solutions.

---

### Tips for Success

- Read the comments carefully — they explain exactly what you need to do.
- If you get stuck for more than 10 minutes, **google the method** (not the solution).
- Write your own code even if it’s not perfect. The goal is understanding, not perfect code.
- You can always come back later and improve your solution.
- Celebrate small wins! Every exercise you complete yourself is progress.

---

### Final Reminder

This is **your chance to make the basics stick**.  
The more you struggle and solve it yourself now, the easier everything later will feel.

You’ve got this! 💪  
Happy coding!
